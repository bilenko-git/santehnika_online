Control
На странице есть блок внутри которого имеется ссылка и блок с контентом, нужно чтобы при клике по ссылке переключалась видимость блока с контентом. Реализация должна быть объектно-ориентированная, требуется написать класс (или функцию конструктор ~es5), конструктор которого будет принимать селектор родительского dom-блока, а метод инициализации будет в себе инкапсулировать алгоритм необходимый для обеспечения требуемой функциональности.
